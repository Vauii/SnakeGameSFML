Общая информация:

    Проект посвящён реализации игры Snake. Игра представляет собой классическую "Змейку" с одним яблоком на поле,
    с подсчётом очков и паузой. Управление производится с клавиатуры, для реализации графики использован модуль SFML.

Дополнительный функционал:

    Использованы:

        1) библиотеки vector, deque для удобной работы со структурой самой змейки и т.д.
        2) библиотеки random и ctime для генерации псевдослучайных позиций яблока
        3) модуль SFML для отображения графики и управления игрой с клавиатуры
        4) библиотека string для работы с текстом

    В качестве паттерна проектирования для реализации структуры кода игры был выбран паттерн Model-View-Controller
